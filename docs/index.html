Agent-to-agent economic infrastructure on Stellar. Autonomous AI agents discovering services, negotiating prices, and settling payments — on-chain, in real time.

What is this? A payment gateway that lets AI agents discover services, pay for them via Stellar, and prove delivery — all in under 5 seconds. Any agent framework plugs in via HTTP. One process handles discovery, dual-protocol payments (x402 + MPP), spending governance, federation identity, fiat on-ramp, and fleet management.

Why Stellar? $0.000003 per transaction. 5-second finality. Path payments (pay in any asset, receiver gets theirs). Federation (human-readable addresses). Built for exactly this.

Is it real? 4 autonomous agents (Atlas, Sage, Pixel, Quant) have been transacting since March 31. Every tx hash below is verifiable on Stellar Expert.
